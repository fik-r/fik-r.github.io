Kenalan dengan npm - Part 1

2020-04-02

Buat kalian yang belum tau, npm itu singkatan dari node package manager. Salah satu tool yang otomatis terinstal ketika kalian menginstal Node.js yang fungsinya untuk mengatur package-package dependensi yang kita gunakan dalam project kita. Kembarannya kayak si composer yang untuk php, pip untuk python, dll. Di part ini gua ga akan bahas detail tentang npm ya, sekilas aja tentang commandnya seperti cara nerapin, install, remove, dll.

Buat kalian yang belum install Node.js dan npm, install dulu deh. Kalian bisa lihat artikel ini bagi yang ga ngerti cara installnya - cara-install-node-js

Quickstart

Menerapkan npm kedalam proyek yang sudah ada

Kalian enter terus aja gpp nanti akan keisi dengan nilai default. kalau udah yes, package.json akan otomatis tergenerate di projectmu.

Menginstall npm packages

Setelah jalanin perintah diatas akan ada folder baru yang namanya node_modules. Isi folder ini ya kumpulan package yang ada di package.json. Fungsi perintah tadi buat install semua package yang ada di dependencies pada package.json.

Kalau mau install spesifik package yang ada di npmjs.com.

Menghapus npm package dari project

Secara otomatis package akan terhapus dari folder node_modules sekaligus di dependencies pada package.json.
